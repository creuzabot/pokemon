# Pokemon API Client

Script Python simples para buscar informações de Pokémons via [PokéAPI](https://pokeapi.co).

## Uso

```bash
python pokemon.py pikachu
python pokemon.py charizard
python pokemon.py 25
```

## Exemplo de saída

```
Nome:        Pikachu
ID:          #25
Tipos:       electric
Altura:      0.4 m
Peso:        6.0 kg
Habilidades: static, lightning-rod
```

## Dependências

```bash
pip install requests
```
